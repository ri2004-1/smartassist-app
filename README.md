# 🧠 SmartAssist — AI-Powered Web App

A full-stack AI web application with real-time chat, 
text summarization and voice interface.

## 🔴 Live Demo
👉 https://smartassist-app.vercel.app

## ✨ Features
- 💬 Real-time AI Chat
- 📝 Text Summarizer (4 modes)
- 🎙️ Voice Input & Output
- 🖼️ Image Analysis (coming soon)

## 🛠️ Tech Stack
- **Frontend:** React.js, HTML, CSS
- **Backend:** Node.js, Express.js
- **AI:** Groq AI API (Llama 3.3)
- **Deployment:** Vercel + Render
- **Version Control:** Git + GitHub

## 🚀 How to Run Locally

### Frontend
cd smartassist-app
npm install
npm start

### Backend
cd smartassist-backend
npm install
npm start
